Documentation for Movie API

URL Endpoints

Request URL HTTP Method Request Body Data Format Response Body Data Format
Return all movies /movies GET None A JSON object holding data about all movies
Return data (description, genre, director, etc.) about a single movie by title /movies/:movieTitle GET None A JSON object holding data about the movie
Return data about a genre (description) by name/title /movies/genre/:genreName GET None A JSON object holding data about the genre
Return data about a director (bio, birth year, death year) by name /movies/directors/:directorName GET None A JSON object holding data about the director
Allow new users to register /users GET A JSON object request from the client A JSON object of the new user
Allow users to update their user info (username) /users/:id PUT A JSON object with updated values for a user A JSON object of the updated user
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST A JSON object of a user and the movie they want added to their favorites A message that the movie has been successfully added
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE A JSON object of a user and the movie they want to remove from their favorites A message that the movie has been successfully removed
Allow users to deregister /users/:id DELETE A JSON object of a user by that includes their id A message that the user has been successfully deleted
